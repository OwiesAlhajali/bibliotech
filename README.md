## About
Bibliotech is an Android app that lets users search, browse,
and explore books using the Google Books API. Built as a 
university project by a team of three.

## Features
- 🔐 Register & Login (Firebase Auth)
- 🔍 Search books by title, author, or publisher
- 📖 View detailed book information
- 🏛️ Browse by publisher

## Tech Stack
| Layer      | Technology              |
|-----------|-------------------------|
| Language  | Kotlin                  |
| UI        | Jetpack Compose         |
| Auth      | Firebase Authentication |
| API       | Google Books API        |
| Network   | Retrofit + OkHttp       |
| DI        | Hilt                    |
| Arch      | MVVM + Clean Arch       |

## Note
This project is part of our Mobile Development course.
demonstrates Kotlin on Android.
